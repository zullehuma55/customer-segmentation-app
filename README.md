# customer-segmentation-app

## Tech Stack
- Python
- Streamlit
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Dataset
The dataset used is `mall_customers.csv`, which contains:
- Customer ID
- Gender
- Age
- Annual Income
- Spending Score

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
